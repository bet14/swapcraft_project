Project Scope

Focused Phase 1 Scope With Clear Boundaries

05
In Scope
Responsive web marketplace
Product and material swap system
Creator profiles and portfolios
Secure payment gateway
Order management and tracking
Reviews and ratings

Custom order requests
Admin dashboard and moderation
Seller analytics dashboard

Out of Scope for Phase 1
  • Native iOS / Android mobile application.
  • Physical shipping or logistics management.
  • Cryptocurrency or blockchain-based payments.
  • AI-powered product recommendation engine.
  • Multi-language or multi-currency support.
  • Third-party seller API integrations.
Scope Principle

Protect the MVP baseline while preserving the features that create differentiation: swap, community, identity, and trust.
